# Конкурс на выполнение Android задачи:
Нужно в текущем репозитории добавить диалог описания и поменять верстку для отображения описания.  
Для отображения диалога описания использовать AlertDialog

## Порядок проведения:
1 - Клонируете себе репозиторий;  
2 - Выполняете задачу;  
3 - Присылаете PullRequest;

PullRequest'ы просматриваются в том порядке, в котором их пришлют.  
Первый кто выполнит задачу без критических недостатков будет победителем и заберет приз.
